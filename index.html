Welcome to the workshop Android for beginners

What do you need for this workshop ?

  • Android Studio: http://developer.android.com/sdk/index.html
    • It is possible that Android studio starts complaining about java jdk, install this jdk from the link Android studio provides. After the installation of java jdk, you have to restart the Android Studio installation.
  • Genymotion (with virtualbox): https://www.genymotion.com/#!/store (free version: Registration needed).
    • Adding an emulator: Start genymotion -> Choose: Add -> Choose a device (Samsung galaxy s3 for example) -> Click 2 times next and the emulator will be created.
    • Starting an Emulator: Start genymotion -> Select the emulator and click run

Structure of the workshop

  • 40 minutes explanation about the structure of an Android project:
    • Explanation about all files and folders
    • Layout
    • Java code to interact with the layout
  • 20-Minute break
  • 1 hour making a simple game

Assignments

Part 1

  • Change the logo of the app. (Zoek op internet een png)
    • Add an image to the drawable or mipmap folder
    • Set this image as icon in the Manifest
  • Make a GameActivity that loads the layout activity_game.
    • Make a new class and let it extend AppCompatActivity
    • Example from MainActivity:public class MainActivity extends AppCompatActivity
  • Add a button so you can go from MainActivity to GameActivity.
    • Declare the button in the layout: activity_main.xml
    • Add an onClickListener in MainActivity.java

Part 2

  • Create a new instance of the game class
    • Game game = new Game(this);
  • Add all pointsButtons to the game (id's are buttonPoints0..9)
    • Use findViewById() to get the reference to the 10 buttons
    • Use the methode addPointsButton(Button button)
  • Set the AnswerButtons to the game
    • Use findViewById() to get the reference to the 10 buttons
    • Use the methode setAnswerButtons(Button button)
  • Set the textview of the time to the game
    • Use the methode setTimerTextView(TextView textView)
  • Set the textview of the question to the game
    • Use the methode setQuestionTextView(TextView textView)
  • Set the onClickListeners of the buttons
    • Use the methode checkIfRightAnswered(String string)
    • This string should be the text of the button
    • Use the methode button.getText().toString() #### Part 3
  • Add a landscape layout for activity_game (buttons alongside each other instead of on top)
  • Add a menu with different layout for portait and landscape modus
  • When clicked on 1 of the menu-items, the game should restart. (game.restart())

Part 4 (Difficult)

  • Set the onWinListener
  • Make sure the game doesn't restart on turn (onConfigCHanged ...)
  • Add a TextView in the layout that shows the level
