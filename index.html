Welcome to the workshop Android for beginners

What do you need for this workshop ?

  • Android Studio: http://developer.android.com/sdk/index.html
    • It is possible that Android studio starts complaining about java jdk, install this jdk from the link Android studio provides. After the installation of java jdk, you have to restart the Android Studio installation.
  • Genymotion (with virtualbox): https://www.genymotion.com/#!/store (free version: Registration needed).
    • Adding an emulator: Start genymotion -> Choose: Add -> Choose a device (Samsung galaxy s3 for example) -> Click 2 times next and the emulator will be created.
    • Starting an Emulator: Start genymotion -> Select the emulator and click run

Structure of the workshop

  • 40 minutes explanation about the structure of an Android project:
    • Explanation about all files and folders
    • Layout
    • Java code to interact with the layout
  • 20-Minute break
  • 1 hour making a simple game
